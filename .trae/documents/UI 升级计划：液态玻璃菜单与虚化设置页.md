# UI 升级计划：液态玻璃菜单与虚化设置页

本计划旨在根据您的视觉参考图，将应用 UI 升级为 iOS 26 概念风格。

## 1. 底部菜单重构 (BottomMenuView)
将静态的磨砂玻璃胶囊升级为**可滑动的液态玻璃**效果，参考 Apple Music 概念设计。

*   **滑块逻辑 (`Sliding Selection`)**：
    *   创建一个高亮的“液态光标”，它会跟随用户的选择在“计时器”和“设置”之间滑动。
    *   使用 `matchedGeometryEffect` 实现平滑的形态变换动画。
    *   添加弹簧动画 (`spring`)，模拟液体的粘性物理反馈。

*   **液态玻璃材质 (`Liquid Glass Material`)**：
    *   **背景层**：保持深色磨砂 (`ultraThinMaterial`)，但增加动态的光影流转效果。
    *   **高亮层**：为选中的 Tab 添加流动的高光边框或背景，模拟光线穿过弯曲玻璃的效果。
    *   **图标**：选中的图标会稍微放大并带有微弱的发光效果。

## 2. 设置页面视觉优化 (SettingsView)
实现顶部**渐变虚化遮罩**，参考 iOS 列表的顶部过渡效果。

*   **顶部虚化 (`Blur Mask`)**：
    *   在 `ZStack` 最顶层添加一个高度较小的区域。
    *   使用 `LinearGradient` 配合 `mask`，实现从顶部的“完全模糊”到下方的“清晰”渐变过渡。
    *   这会让滚动的列表内容在顶部优雅地“消融”，而不是生硬地被截断。

*   **背景一致性**：
    *   确保虚化层与全局 `darkGray` 背景无缝融合。

## 3. 执行步骤
1.  **重构 `BottomMenuView.swift`**：实现滑动游标和液态视觉效果。
2.  **修改 `SettingsView.swift`**：添加顶部渐变虚化遮罩。
3.  **微调**：调整动画曲线和阴影参数，确保质感符合“iOS 26”的高级审美。

完成这些更改后，您的应用将拥有极具未来感和流动感的交互体验。
